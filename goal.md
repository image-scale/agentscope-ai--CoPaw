# Goal

## Project
CoPaw — a python project.

## Description
CoPaw is a personal AI assistant platform that enables users to run LLM-powered agents locally or in the cloud. It provides a unified interface to multiple LLM providers (OpenAI, Anthropic, Ollama, local models via llama.cpp), with features including provider management, model activation/persistence, local model downloading and server management, environment variable handling, and a CLI tool for initialization and operation.

## Scope
- 15 production source files to implement
- 10 test files to write
- Core functionality: provider system, provider manager, local model management, configuration, CLI
